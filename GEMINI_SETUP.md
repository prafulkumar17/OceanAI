# Using Google Gemini API

This project uses Google's Gemini API for AI-powered document analysis.

## Getting Your API Key

1. Visit: https://makersuite.google.com/app/apikey
2. Sign in with your Google account
3. Click "Create API Key"
4. Copy your API key

## Setup

1. Edit `backend/.env` file
2. Add your Gemini API key:
   ```
   GEMINI_API_KEY=your_api_key_here
   ```

## Benefits of Gemini

- ✅ **Free tier available** - Generous free usage limits
- ✅ **No credit card required** for free tier
- ✅ **Fast and reliable** - Google's infrastructure
- ✅ **Great for document analysis** - Excellent text understanding

## API Limits

- Free tier: 60 requests per minute
- Paid tier: Higher limits available

## Model Used

- **gemini-pro**: Google's latest general-purpose model
- Optimized for text analysis and summarization

## Troubleshooting

If you get API errors:
1. Verify your API key is correct
2. Check you're not exceeding rate limits
3. Ensure internet connection is active
4. Try regenerating your API key if issues persist
